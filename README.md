# ポモドーロタイマー

集中と休憩を繰り返すための、軽量なポモドーロタイマーです。`index.html` 1ファイルだけ。インストールもビルドも不要。

## 使い方
1. `index.html` をブラウザで開く
2. 「集中」「休憩」の分数を設定
3. 「開始」を押す
4. 時間になるとアラームが鳴る → **「アラームを止める」を押すまで鳴り続ける**
5. 止めると自動で次（集中⇄休憩）に切り替わる

止めるまでの放置時間は「オーバー時間」として記録され、今日の合計（集中／休憩／オーバー時間／ポモドーロ回数）が画面下に出ます。日付が変わると自動でリセットされます。

**ショートカット**: `Space` 開始/一時停止 ・ `Esc` アラーム停止 ・ `R` リセット

## メモ
- 裏タブでも時間がずれません。タブのタイトルに残り時間が出ます。
- デスクトップ通知（許可制）で、別のことをしていても気づけます。
- データはブラウザ内（localStorage）に保存。端末・ブラウザ間では共有されません。
